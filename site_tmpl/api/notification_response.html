Parameter Value Description
id string Unique reference id for the specific notification
format string Defines how the notification should be displayed in the view
Options: alert, notification
type string Defines the type/purpose of the message
Options: info, advisory, warning
class int Defines the level of control the user has over the notification and may determine how it is rendered

Class I: Notification/alert cannot be silenced. Reserved for site-wide notifications only.
Class II: Alerts are automatically silenced once viewed. Notifications cannot be silenced by user. Default.
Class III: User can manually silence notification/alert.
title string Title to be displayed with alert (for alerts only)
message string The notification message
expires datetime The notification's expiration date
